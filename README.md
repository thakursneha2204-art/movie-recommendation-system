# Movie Recommendation System

This is a Streamlit-based movie recommendation web application.
The app provides movie recommendations using a pre-built recommendation logic.

## Technologies Used
- Python
- Streamlit

## Description
The recommendation logic is based on an existing model/approach,
which is integrated into a Streamlit interface for user interaction.

This project was developed as a learning exercise with guidance
and reference resources.
